Show all of your in-flight PRs on a Figma-like canvas. Supports 
 * stacked PRs (through [Git Spice](https://abhinav.github.io/git-spice/))
 * Buildkite statuses/specific failing checks
 * Trunk.io status reading
 * Light/dark mode

## Requirements
`gh` and `bk` CLIs installed
`npm` and `uv`

## Installation
`cd web && npm run build`

## Running
`uv run app.py`
